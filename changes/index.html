Change Log

Carneades Web App (Version 0.6, 12 December 2012)

• Now a multiuser Web application, with a three-tier architecture, including a relational database backend.
• New argument evaluator, based on ASPIC+ and Dung abstract argumentation frameworks. Cyclic argument graphs are now handled.
• Argument maps are now represented using SVG.
• New XML format for exporting and interchanging argument graphs, called the Carneades Argument Format (CAF). Replaces LKIF from the ESTRELLA project.
• New rule language, based on Lisp s-expressions, for representing policies and argumentation schemes. Replaces the LKIF rule language, which used XML.

Carneades Editor (Version 1.0.2 , 11 October 2011)

• Various bug fixes

Carneades Editor (Version 1.0.0, 31 Mar 2011)

• keyboard shortcut, ctrl+Q, for quitting the application
• update of the installation instructions
• context menu for changing premises types and polarities
• context menu for changing arguments direction
• changed terminology of the sort in the Find Positions Assistant
• LKIF extended to handle domains and existential quantifiers
• better MacOS integration
• various bug fixes

Carneades Editor (Version 0.96, 15 Feb 2011)

• Fixed a layout bug

Carneades Editor (Version 0.95, 1 Feb 2011)

• Support for formalizing statements, using the classes and properties of an OWL ontology
• Command-key equivalents for some common commands
• The argumentation scheme of each argument, if one has been assigned, is displayed in argument maps.
• User preferences can be modified, using the the Edit/Preferences dialog. Currently, two preferences can be specified: 1) the LKIF file containing the model of the user’s preferred argumentation schemes, used by the “instantiate scheme” assistant; and 2) A directory to use as a library of LKIF and OWL knowledge bases. When relative pathnames are used to import OWL and LKIF files into an LKIF file, the files are searched for first in the same directory as the LKIF file which imported the files and then in the library directory specified in the user preferences dialogue. Libraries may be organized hierarchically, using file system folders.

Carneades Editor (Version 0.90, 9 Dec 2010)

• Support for constructing arguments by instantiating argumentation schemes represented as rules in LKIF;
• Support for constructing arguments from in LKIF and OWL knowledge bases; and
• Support for finding positions, i.e. minimals set of goal statements, using abduction, which if proven acceptable would make a goal statement in or out.
